This project analyzes Netflix Movies and TV Shows using Python and Power BI.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Power BI

## Analysis Performed
- Data Cleaning
- Genre Analysis
- Rating Analysis
- Country-wise Content Distribution
- Top Directors and Actors
- Word Cloud Visualization

Interactive Power BI dashboard for exploring Netflix content trends.
